Media Management

How to Add Photos & Videos

Three simple ways to upload and display your media on this website.

01

Replace Image URLs

Open the HTML file in any text editor (Notepad, VS Code). Find the <img src="..."> tags and replace the URL with your own image link from Google Drive, Dropbox, or any image host.

Example: src="https://your-image-link.jpg"

02

Use Local Files

Put your photos/videos in the same folder as this HTML file. Then update the src attribute to the filename only.

Example: src="wedding-shot.jpg"

For videos, use the <video> tag with src="myvideo.mp4".

03

Embed YouTube / Vimeo

Copy the embed link from YouTube or Vimeo and paste it inside an <iframe> in the gallery section of the HTML.

Example: <iframe src="https://youtube.com/embed/VIDEO_ID"></iframe>

Try It Live — Drag & Drop Here

Drag your photos or videos onto this area to preview them instantly on the page.
